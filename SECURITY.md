# Security Policy

## Scope

`padv` is a security testing tool designed for use against local and sandbox targets only. It does not scan external targets.

## Reporting Vulnerabilities

If you discover a security vulnerability in `padv` itself, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Email security@mayflower.de with a description of the vulnerability
3. Include steps to reproduce if possible

We will acknowledge receipt within 3 business days and aim to provide a fix or mitigation plan within 14 days.

|---------|-----------|
